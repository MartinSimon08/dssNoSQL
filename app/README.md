Cómo ejecutar::

desde la carpeta app. Backend y Frontend no estan en carpetas diferentes

Iniciar MongoDB (servicio activo).

Levantar el backend:

node server/index.js
tiene que retornar:

Servidor en puerto 4000
Mongo conectado

Corre en: http://localhost:4000

Levantar el frontend:

npm run dev
Abrir: http://localhost:5173
